Option C — Hybrid (Bank Pulse)

Slim strip header with the brand label and a one-line summary. Below it: a Bank Pulse panel — live case counter with a 24h sparkline, top deviations across all engagements, and a rolled-up Elastic Operations triangle. Engagement cards live below the pulse.
+ Communicates "process intelligence cockpit" — distinct from generic SaaS + Live pulse + cross-engagement insights feel valuable every visit + Surfaces deviations & $ leak at the top + Most differentiated from Celonis-style
− Most ambitious build (~4-5 days) − Needs cross-engagement aggregation API (some new server work) − "Live" sparkline needs real-time data or convincing approximation

Option D — Process: Lean

Compact single-screen layout. Single-row process header (LOB / name / client / status / settings). 4 KPI cards. One-line sub-process capability heatmap (11 cells colour-coded by maturity). 3 large discovery-phase tiles. Metrics + capabilities form is hidden behind "⚙ Edit details" — opens as a modal/drawer.
+ Fits in one viewport · no scroll + Capability heatmap is the single biggest improvement — instant glance at where the gaps are + Discovery tiles get the visual weight they deserve + Settings hidden but discoverable
− Editing metrics requires a modal click − Heatmap labels can get tight at 11+ sub-processes
